API clients

Import RouterBase before writing code

The public RouterBase API collections repository contains request files for OpenAPI, Postman, Bruno, and Insomnia.

Format Path Use case
OpenAPI openapi/routerbase.openapi.json Documentation, SDK experiments, contract review
Postman postman/routerbase.postman_collection.json Manual request testing
Bruno bruno/RouterBase Git-friendly API collections
Insomnia insomnia/routerbase.insomnia.json Workspace import and request exploration

Environment

ROUTERBASE_BASE_URL=https://routerbase.com/v1
ROUTERBASE_API_KEY=your-secret-key

Smoke test order

  1. Run GET /models to confirm authentication.
  2. Run POST /chat/completions with a short prompt.
  3. Move the same request body into your application code.
